# Arduino valdymas su Python naudojant OOP  

Šis projektas naudoja **objektinį programavimą (OOP)**, kad valdytų Arduino mikrokontrolerį ir jo komponentus (LED, mygtukas) teisiogiai iš Phyton aplinkos.  

## Naudojama  

- Python 3.14.3
- Mikrokontroleris: Arduino Uno R3
- Biblioteka: `pyFirmata`

## Pradžia  

1. Instrukcija: [Diegimo žingsniai](./SETUP.md)  

2. Paleidimas: `python {failo pavadinimas}.py`.  
*Svarbu: prieš paleidžiant komandą, įsitikinti, kad terminalas atidarytas toje vietoje, kur yra `{failo pavadinimas}.py` failas.*
