# Arduino valdymas su Python naudojant OOP  

Šis projektas naudoja **objektinį programavimą (OOP)**, kad valdytų Arduino mikrokontrolerį ir jo komponentus (LED, mygtuką, potenciometrą) teisiogiai iš Phyton aplinkos.  

## Naudojama  

- Python 3.14.3
- Biblioteka: `pyFirmata`
- Mikrokontroleris: Arduino Uno R3
- 3 - LED
- 1 - Potenciometras
- 1 - Mygtukas

## Pradžia  

1. Instrukcija: [Diegimo ir jungimo žingsniai](./SETUP.md)  

2. Paleidimas: `python {failo pavadinimas}.py`.  
*Svarbu: prieš paleidžiant komandą, įsitikinti, kad terminalas atidarytas toje vietoje, kur yra `{failo pavadinimas}.py` failas.*
